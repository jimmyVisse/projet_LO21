Projet LO21 UTC Semestre P14

Réalisé par Jimmy Visse et Sarah Ouro-Yondou

But: Réaliser une application permettant de gérer le parcours d'un étudiant à l'UTC.
De plus, l'application devra permettre la gestion des UVs ainsi que des cursus présents à l'Université de Technologie de Compiègne.

Ce projet sera réalisé via le langage C++ avec le framework Qt
